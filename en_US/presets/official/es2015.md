### Installation

```sh
$ npm install --save-dev babel-preset-es2015
```

### Usage

#### Via `.babelrc` (recommended)

**`.babelrc`**

```json
{
  "presets": ["es2015"]
}
```

#### Via CLI

```sh
babel script.js --preset es2015
```

#### Via Node API

```js
require("babel-core").transform("code", {
  presets: ["es2015"]
});
```
